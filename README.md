# devLab101

This repo contains reusable UI interface projects built with React.js.

## Getting Started with Vite + React

This project was bootstrapped using `npm create vite@latest -- --template react`.

### Available Commands

In the project directory, you can run the following commands in your terminal:

#### `npm run dev`

Runs the app in development mode.
Open [http://localhost:5173](http://localhost:5173) to view it in your browser. The page will instantly reload when you save changes to your code.

#### `npm run build`

Builds the app for production.
This packages your React code into a `dist` folder, optimizing the files for the best possible speed and performance when you are ready to host the website online.

#### `npm run preview`

Lets you locally preview the production build you created with the `npm run build` command, allowing you to test exactly what users will see before you launch.
